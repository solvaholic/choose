# choose
A small shell script to help you choose among N options

```text
% ./choose.py 
Enter items to choose from (one per line). Type 'done' when finished:
sleep
write choose.py
do the next thing on your to-do list
done
You are considering the following items:
A: sleep
B: write choose.py
C: do the next thing on your to-do list
Would you rather (A) sleep or (B) write choose.py? B
Would you rather (A) sleep or (C) do the next thing on your to-do list? A
Would you rather (B) write choose.py or (C) do the next thing on your to-do list? B
write choose.py: 2
sleep: 1
```
